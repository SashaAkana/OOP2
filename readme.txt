Система управления библиотекой включает три основных класса: Book, Reader и Library. Каждый из этих классов имеет свои свойства и методы, которые позволяют управлять книгами и читателями. Давайте рассмотрим каждый класс отдельно.

1. Класс Book
Свойства
title: название книги (строка).
author: автор книги (строка).
year: год издания книги (целое число).
isAvailable: доступность книги в библиотеке (булево значение, по умолчанию true).
Методы
getTitle(): Возвращает название книги.
getAuthor(): Возвращает автора книги.
getYear(): Возвращает год издания.
getAvailability(): Возвращает доступность книги (доступна или нет).
setAvailability($status): Изменяет статус доступности книги (принимает булево значение). Если книга выдана, устанавливается значение false; если возвращена — true.
Пример использования
php
$book = new Book("1984", "Джордж Оруэлл", 1949);
echo $book->getTitle(); // выводит "1984"
2. Класс Reader
Свойства
name: имя читателя (строка).
email: адрес электронной почты читателя (строка).
borrowedBooks: массив книг, которые читатель взял (по умолчанию пустой массив).
Методы
getName(): Возвращает имя читателя.
getEmail(): Возвращает электронный адрес читателя.
borrowBook($book): Метод, который позволяет читателю взять книгу. Если книга доступна, она добавляется в массив взятых книг, а её статус доступности меняется на false.
returnBook($book): Метод, который позволяет читателю вернуть книгу. Книга удаляется из массива взятых книг, и её статус доступности меняется на true.
Пример использования
php
$reader = new Reader("Иванов Иван", "ivanov@example.com");
$reader->borrowBook($book); // Читатель берет книгу
3. Класс Library
Свойства
books: массив всех книг в библиотеке.
readers: массив всех читателей.
Методы
addBook($book): Добавляет книгу в библиотеку. Книга становится частью массива books.
addReader($reader): Добавляет читателя в систему. Читатель становится частью массива readers.
findBook($title): Ищет книгу по названию. Если книга найдена, возвращает объект книги; если нет — возвращает null.
listBooks(): Выводит список всех книг в библиотеке, показывая их названия и авторов.
Пример использования
php
$library = new Library();
$library->addBook($book); // Добавляем книгу в библиотеку
$library->listBooks(); // Выводит список книг
4. Файл index.php
В этом файле производится интеграция всех классов и демонстрация их работы:

Подключение классов: Все необходимые классы подключаются с помощью include.
Создание экземпляров: Создаются объекты для библиотеки, книг и читателей.
Добавление книг и читателей в библиотеку: Используются методы addBook() и addReader().
Операции с книгами: Читатель может взять книгу, используя метод borrowBook(), и вернуть её с помощью метода returnBook().
Вывод списка книг: Список всех книг выводится с помощью метода listBooks().
